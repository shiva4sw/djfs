1. Create Django Project
Start Virtual Environment(venv\Scripts\activate)
django-admin startproject blog_project
Navigate into it: 
cd blog_project
2. Create Blog App
python manage.py startapp blog
Add `'blog'` to INSTALLED_APPS in settings.py.
3. Create Post Model
In blog/models.py: 

from django.db import models
from django.contrib.auth.models import User

class Post(models.Model):
    title = models.CharField(max_length=200)
    content = models.TextField()
    author = models.ForeignKey(User, on_delete=models.CASCADE)
    created = models.DateTimeField(auto_now_add=True)
4. Create ModelForm
In blog/forms.py: 

from django import forms
from .models import Post

class PostForm(forms.ModelForm):
    class Meta:
        model = Post
        fields = ['title', 'content']
5. Create Views
In blog/views.py: 

from django.shortcuts import render, redirect
from .models import Post
from .forms import PostForm
from django.contrib.auth.decorators import login_required

def post_list(request):
    posts = Post.objects.all().order_by('-created')
    return render(request, 'blog/post_list.html', {'posts': posts})

@login_required
def post_create(request):
    if request.method == 'POST':
        form = PostForm(request.POST)
        if form.is_valid():
            post = form.save(commit=False)
            post.author = request.user
            post.save()
            return redirect('post_list')
    else:
        form = PostForm()
    return render(request, 'blog/post_form.html', {'form': form})
6. Create URLs
In blog/urls.py: 

from django.urls import path
from . import views

urlpatterns = [
    path('', views.post_list, name='post_list'),
    path('new/', views.post_create, name='post_create'),
]
In blog_project/urls.py: 

from django.contrib import admin
from django.urls import path, include
from django.contrib.auth import views as auth_views

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('blog.urls')),
    path('login/', auth_views.LoginView.as_view(), name='login'),
    path('logout/', auth_views.LogoutView.as_view(), name='logout'),
]
7. Create Templates
  • base.html: common layout with login/logout links
  • post_list.html: list of blog posts
  • post_form.html: form for creating a post
8. Configure Login Redirects
In settings.py: 

LOGIN_REDIRECT_URL = '/'
LOGOUT_REDIRECT_URL = '/'
9. Migrate & Run

python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
