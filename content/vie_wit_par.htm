Step 1: Define a view with multiple parameters in views.py
Example:
from django.http import HttpResponse

def show_profile(request, name, age):
    return HttpResponse(f"Name: {name.capitalize()}, Age: {age}")
Step 2: Add a URL pattern with multiple parameters in urls.py of your app
Example:
from django.urls import path
from . import views

urlpatterns = [
    path('profile/<str:name>/<int:age>/', views.show_profile, name='profile'),
]
