1. What are Template Tags?
Django template tags are special syntax enclosed in {% ... %} used to implement logic (like loops, conditionals) inside templates(HTML files).
2. Variable Tags
Use {{ variable_name }} to display the value of a variable passed from a view.
Example: {{ user.username }}
3. Common Built-in Template Tags
  • {% if %} - Used for conditional logic.
  • {% for %} - Used for looping over a list.
  • {% block %} - Marks a block in base templates for inheritance.
  • {% extends %} - Inherits from a parent template.
  • {% include %} - Includes another template file.
  • {% csrf_token %} - Protects against CSRF attacks in forms.
  • {% url %} - Resolves view names into URLs.
  • {% load %} - Loads custom tags or filters.
4. Example: Conditional & Loop Tags
{% raw %}
{% if user.is_authenticated %}
  Hello, {{ user.username }}
{% else %}
  Hello, guest!
{% endif %}


{% for item in items %}
  {{ item }}
{% endfor %}

{% endraw %}
5. Example: Template Inheritance
base.html
{% raw %}
<html>
  <body>
    {% block content %}{% endblock %}
  </body>
</html>

child.html
{% extends "base.html" %}
{% block content %}
  <h1>Welcome to the site!</h1>
{% endblock %}
{% endraw %}
