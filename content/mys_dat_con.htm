1. Install MySQL Client Library
Install the Python MySQL client: 
pip install mysqlclient
If using Windows and facing issues, you can try: 
pip install pymysql
2. Create a MySQL Database
Using MySQL terminal or MySQL Workbench: 

CREATE DATABASE myproject;
3. Configure settings.py
In your Django projectâ€™s settings.py, update the DATABASES section: 

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'myproject',
        'USER': 'myuser',
        'PASSWORD': 'mypassword',
        'HOST': 'localhost',
        'PORT': '3306',
    }
}
If using pymysql instead of mysqlclient, add this at the top of settings.py: 

import pymysql
pymysql.install_as_MySQLdb()
4. Run Migrations
Apply your Django models to the MySQL database: 

python manage.py makemigrations
python manage.py migrate
5. Run Server
python manage.py runserver
6. Troubleshooting
- Ensure MySQL service is running.
- Use the correct host and port.
- For MySQL 8+, you may need to set auth_plugin=mysql_native_password for your user.
