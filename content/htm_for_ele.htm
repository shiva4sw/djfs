1. <form>
The container element for form controls. Attributes: action, method (GET or POST), autocomplete, etc.
2. <input>
HTML5 introduced many new type values for <input>:
  • text – Single-line text input
  • email – Validates email format
  • url – Validates URLs
  • tel – Telephone numbers
  • number – Numeric input with min/max
  • range – Slider control
  • date – Date picker
  • time – Time picker
  • datetime-local – Local datetime picker
  • month – Month picker
  • color – Color picker
  • search – Styled for search input
3. <textarea>
For multi-line text input.
4. <select> and <option>
Dropdown list of options. Use multiple to allow multiple selections.
5. <datalist>
Provides autocomplete suggestions for an <input> element. 

<input list="browsers">
<datalist id="browsers">
  <option value="Chrome">
  <option value="Firefox">
</datalist>
6. <fieldset> and <legend>
Group related fields together with a label.
7. <label>
Defines a label for an input. Use for attribute to bind it to an id.
8. <button>
Can be submit, reset, or just a button to trigger JS.
9. Validation Attributes
  • required – Field must be filled out
  • pattern – Regex validation
  • min, max – Range limits
  • step – Specifies legal number intervals
  • maxlength, minlength – Text length limits
10. Autofocus, Placeholder, Disabled
  • autofocus – Automatically focuses on the input
  • placeholder – Displays hint text
  • disabled – Makes field uneditable
