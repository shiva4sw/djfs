Creating Simple App

Step 1: Navigate to your Django project folder
Use the command: cd myproject
Step 2: Create a new app
Run the command: python manage.py startapp myapp
Replace myapp with your desired app name.
Step 3: Register the app in settings.py
Open myproject/settings.py and add 'myapp' to the INSTALLED_APPS list.
Step 4: Open your app's views.py
Define a function that returns a response. For example: 

from django.http import HttpResponse

def home(request):
    response_str = "Hello, Django!"  
    return HttpResponse(response_str)
Use HTML tags in response_str to display pages
Step 5: Create urls.py in your app (if not already created)
Add the following code: 

from django.urls import path
from . import views

urlpatterns = [
    path('', views.home, name='home'),
]
Step 6: Include the app URLs in the projectâ€™s urls.py
In myproject/urls.py, include your app's URLs: 

from django.contrib import admin
from django.urls import path, include

urlpatterns = [
    path('admin/', admin.site.urls),
    path('', include('myapp.urls')),
]
Step 7: Run the server
Use the command: python manage.py runserver and open http://127.0.0.1:8000/ in your browser to see the output.
