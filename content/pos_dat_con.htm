1. Install PostgreSQL Client Library
Install the Python PostgreSQL client: 
pip install psycopg2-binary
2. Configure settings.py
In your Django projectâ€™s settings.py, update the DATABASES section: 

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'mydb',
        'USER': 'myuser',
        'PASSWORD': 'mypassword',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
4. Run Migrations
Apply your Django models to the PostgreSQL database: 

python manage.py makemigrations
python manage.py migrate
5. Run Server
python manage.py runserver
