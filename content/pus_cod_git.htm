1. Git Login (if not already)

git config --global user.name "username"
git config --global user.email "useremail"
This initializes a new Git repository in your project folder.
2. Add .gitignore file
      
# Python files
__pycache__/
.venv/
venv/

# Django files
staticfiles/
db.sqlite3
3. Add Remote Repository
git remote add origin https://github.com/your-username/your-repo.git
Replace the URL with your actual GitHub repository link. 
git init
4. Add Files

git add .
This stages all files for commit.
5. Commit Changes

git commit -m "Initial commit"
Replace the commit message with a meaningful description.
6. Push Code

git push -u origin main
Use main if your default branch is named `main`. Use master if it's `master`.

You can create and push a new branch:

8. Common Issues
  • Pull from git: git pull origin main --rebase
  • Authentication Error: Use GitHub CLI or personal access token
  • Remote Already Exists: git remote set-url origin <new-url>
