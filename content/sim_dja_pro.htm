Simple Django Project

Step 1: Install Django
Open VS Code->Start Virtual Env and Run the command: pip install django
Step 2: Create a new Django project
Run the command: django-admin startproject myproject
Replace myproject with your desired project name.
Step 3: Navigate to the project directory
Run the command: cd myproject
Step 4: Run the development server
Run the command: python manage.py runserver
Then open your browser and go to http://127.0.0.1:8000/
Step 5: Create a Django app
Run the command: python manage.py startapp myapp
Replace myapp with your desired app name.
Step 6: Add app to project settings
Open myproject/settings.py and add 'myapp' to the INSTALLED_APPS list.
