1. What is JavaScript?
JavaScript is a scripting language used to make web pages interactive. It can update content, validate forms, handle events, and much more.
2. How to Add JavaScript to HTML
  • Inline: <button onclick="alert('Hello!')">
  • Internal: Inside <script> tags in HTML
  • External: In a .js file linked using <script src="file.js">
3. Basic Syntax

let name = "John";
const age = 25;
alert("Hello, " + name);
4. Variables
  • var – old, function-scoped
  • let – modern, block-scoped
  • const – block-scoped, cannot be reassigned
5. Data Types
  • String
  • Number
  • Boolean
  • Array
  • Object
  • null and undefined
6. Functions

function greet(name) {
  return "Hello, " + name;
}
console.log(greet("Alice"));
7. Events
JavaScript reacts to user actions (click, hover, input). 

document.getElementById("btn").onclick = function() {
  alert("Clicked!");
};
8. DOM Manipulation

document.getElementById("title").innerText = "New Title";
9. Conditions and Loops

if (age > 18) {
  alert("Adult");
} else {
  alert("Minor");
}

for (let i = 0; i < 5; i++) {
  console.log(i);
}
10. Arrays and Objects

let colors = ["red", "green", "blue"];
let person = { name: "Alice", age: 22 };
console.log(person.name);
