Steps to Create virtualenv

Step 1: Install virtualenv (optional if using venv)
Run pip install virtualenv if you're using virtualenv instead of the built-in venv (Python 3.3+).
Step 2: Create virtual environment
Use one of the following commands:
- python -m venv env_name (recommended for Python 3)
- virtualenv env_name (if using virtualenv)
Step 3: Activate virtual environment
On Windows: env_name\Scripts\activate
On macOS/Linux: source env_name/bin/activate
Step 4: Install packages
Run pip install package_name inside the activated environment.
Step 5: Deactivate environment
Run deactivate to exit the virtual environment.
