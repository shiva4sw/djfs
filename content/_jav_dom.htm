1. What is the DOM?
DOM stands for Document Object Model. It represents the HTML page as a tree of objects that JavaScript can interact with to change content, style, or structure.
2. Accessing Elements
You can access HTML elements using methods like:
  • document.getElementById("id")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")
3. Changing Content
Use innerHTML or innerText: 

document.getElementById("demo").innerHTML = "Hello, DOM!";
4. Changing Attributes

document.getElementById("image").src = "new-image.jpg";
document.getElementById("link").href = "https://example.com";
5. Changing Styles

document.getElementById("box").style.backgroundColor = "yellow";
6. Event Handling
JavaScript can respond to events like clicks, inputs, etc. 

document.getElementById("btn").onclick = function() {
  alert("Button clicked!");
};
7. Creating Elements

let p = document.createElement("p");
p.innerText = "This is a new paragraph.";
document.body.appendChild(p);
8. Removing Elements

let el = document.getElementById("demo");
el.remove(); // Or: el.parentNode.removeChild(el);
9. Input Handling

let value = document.getElementById("username").value;
alert("Hello " + value);
10. DOM Tree
HTML is represented as a tree of nodes:
  • Document → html → head, body
  • Each element is a node and can be accessed/modified using DOM
