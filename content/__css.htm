1. What is CSS?
CSS stands for Cascading Style Sheets. It is used to style HTML elements — controlling layout, colors, fonts, spacing, and more.
2. Ways to Apply CSS
  • Inline: Inside the HTML tag
    <h1 style="color:red;">Title</h1>
  • Internal: Inside <style> tag in HTML <head>
  • External: In a separate style.css file linked with <link rel="stylesheet" href="style.css"/> tag
3. CSS Syntax

selector {
  property: value;
}
Example: p { color: blue; font-size: 16px; }
4. Selectors
  • * – Universal selector
  • tag – Element selector (e.g., p)
  • .class – Class selector
  • #id – ID selector
  • div > p – Child selector
  • div p – Descendant selector
5. Properties
Common properties:
  • color, background-color
  • font-size, font-family, font-weight
  • margin, padding
  • width, height
  • border, box-shadow
  • display, position, float
6. Box Model
Elements are boxes with: Content → Padding → Border → Margin.
Use box-sizing: border-box; to include padding and border in width/height.
7. Units
  • Absolute: px, cm, mm
  • Relative: %, em, rem, vh, vw
8. Responsive Design
Use media queries to make layouts responsive: 

@media (max-width: 600px) {
  body {
    background-color: lightblue;
  }
}
9. CSS Frameworks
Popular frameworks include:
  • Bootstrap
  • Tailwind CSS
  • Materialize
10. Animations and Transitions

div {
  transition: background-color 0.3s ease;
}
div:hover {
  background-color: yellow;
}
